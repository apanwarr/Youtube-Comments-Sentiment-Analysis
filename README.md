### **Setup & Installation**

### **Create your conda enviroment**

```bash
conda create -n yt-sentiment python=3.10
conda activate yt-sentiment
```

### **Install Required Libraries**
```bash
pip install -r requirements.txt
```

### **Run the Streamlit app**
```bash
streamlit run app.py
```
